# Changelog

## [v0.5.0] - 2026-03-04

### 🔬 Science-Based Improvements

#### 🎵 音響工学

- **1/fゆらぎ打鍵音** — ピッチがブラウニアンウォークで有機的に変化。Math.random()の白色ノイズ特性→1/fフラクタルの自然なメロディへ
- **確率共鳴レイヤー** — 閾値以下の高域ノイズで打鍵音の知覚を向上（NIH確率共鳴理論）
- **バイノーラルビート精緻化** — Zen Drops: 40Hz(γ)→10Hz(α波=創造性)、各プリセットに脳波帯域ラベル追記

#### 🔤 タイポグラフィ

- **行間最適化** — 2.0→1.75に変更（WCAG 1.5x+を満たしつつ密度感向上）
- **字間最適化** — 0.02em→0.01emで日本語の自然な間隔に

#### ✍️ テキストエディタUX

- **タイプライタースクロール** — カーソルを常に画面中央に固定（視線移動排除→フロー維持）
- **Markdownライブプレビュー** — #, ##, ###の見出しを自動フォーマット
- **文字数カウンター** — 左下に極小表示、Ghost Mode中は消滅
- **30秒ソフトフェード** — 無操作30秒でテキスト全体が徐々に消滅、キー入力で即復帰（一時的前頭前皮質低活性化の支援）

## [v0.4.0] - 2026-03-04

### ✨ Feature

- **全プリセットWPMリアクティブ化** — 打鍵速度に応じてデケイ/ピッチ/ブライトネスが動的変化
- **句読点・改行・Backspaceアクセント音** — 文の呼吸を音で表現（ベル/共鳴/逆吸い）
- **プリセット切替クロスフェード** — 2.5秒の滑らかなアンビエント遷移
- **Zen Focus Mode** — タイプ中は現在の段落以外を自動フェードアウト
- **Ghost Mode** — タイプ中にすべてのUI要素が消滅し、文字と背景だけの空間に
- **Pulse Indicator** — タイピングリズムの安定性を画面端のグローで可視化

## [v0.3.0] - 2026-03-03

### ✨ Feature

- 没入感をテーマにした4つの新サウンドプリセット追加（合計10プリセット）
  - **Void** 🕳️ — 虚空に沈み込む極低音ドローン（brownノイズ + 超低域フィルタ）
  - **Forest Temple** 🌿 — 森の神殿で書く。風の音と木霊（pinkノイズ + 中域帯）
  - **Deep Ocean** 🫧 — 深海の静寂。シータ波(4Hz)バイノーラルビートで深い集中へ
  - **Aurora** 🌌 — オーロラの揺らぎ。WPM連動で音が変化し、速く打つほど光が増す
