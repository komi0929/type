# Changelog

## [v0.2.0] - 2026-03-03

### ✨ Feature

- ダークモード / ライトモード切替機能
  - 右上のトグルボタン（☀ / ☾）でワンクリック切替
  - コマンドパレット（Ctrl+K）内にもテーマ切替オプションを追加
  - CSS変数ベースのテーマシステム（スムーズなトランジション付き）
  - localStorage永続化 + システムプリファレンス検出
  - ライトモード: 温かみのある紙色背景（#faf8f5）+ 焦げ茶テキスト
  - ダークモード: 深い黒背景（#0d0d0f）+ クリームテキスト

### ♻️ Refactor

- コマンドパレットを「サウンドプリセット」と「表示」セクションに分離
- ハードコードされたrgba値をCSS変数に統一

## [v0.1.0] - 2026-03-03

### ✨ Feature

- プロジェクト初期セットアップ（Next.js 16 + Tailwind CSS 4）
- ZenCanvasコンポーネント（没入型ライティングエリア）
- CommandPaletteコンポーネント（キーボード操作パレット）
- サウンドエンジン（Web Audio APIベースのタイピング音フィードバック）
- 6種のサウンドプリセット
- ローカルストレージによる自動保存

### 🔧 Config

- 自動記録ワークフロー（`.agents/workflows/auto-record.md`）を追加
