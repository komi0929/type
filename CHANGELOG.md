# Changelog

## [v0.3.0] - 2026-03-03

### ✨ Feature

- 没入感をテーマにした4つの新サウンドプリセット追加（合計10プリセット）
  - **Void** 🕳️ — 虚空に沈み込む極低音ドローン（brownノイズ + 超低域フィルタ）
  - **Forest Temple** 🌿 — 森の神殿で書く。風の音と木霊（pinkノイズ + 中域帯）
  - **Deep Ocean** 🫧 — 深海の静寂。シータ波(4Hz)バイノーラルビートで深い集中へ
  - **Aurora** 🌌 — オーロラの揺らぎ。WPM連動で音が変化し、速く打つほど光が増す

## [v0.2.0] - 2026-03-03

### ✨ Feature

- ダークモード / ライトモード切替機能
  - 右上のトグルボタン（☀ / ☾）でワンクリック切替
  - コマンドパレット（Ctrl+K）内にもテーマ切替オプションを追加
  - CSS変数ベースのテーマシステム（スムーズなトランジション付き）
  - localStorage永続化 + システムプリファレンス検出
  - ライトモード: 温かみのある紙色背景（#faf8f5）+ 焦げ茶テキスト
  - ダークモード: 深い黒背景（#0d0d0f）+ クリームテキスト

### ♻️ Refactor

- コマンドパレットを「サウンドプリセット」と「表示」セクションに分離
- ハードコードされたrgba値をCSS変数に統一

## [v0.1.0] - 2026-03-03

- 6種のサウンドプリセット
- ローカルストレージによる自動保存
